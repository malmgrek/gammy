gammy.arraymapper.lift

lift¶

lift(f)[source]¶

Lift a function

Examples

The main use case is to transform a numeric function to act on array mappers.

import numpy as np

from gammy.arraymapper import x, lift
from gammy.formulae import Scalar

sin = lift(np.sin)
cos = lift(np.cos)
formula = Scalar() * sin(x) + Scalar() * cos(x)
